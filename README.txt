== Description ==

While using [Distributor](https://distributorplugin.com/) along with [Advanced Custom Fields](https://www.advancedcustomfields.com/), this plugin ensures the correct handling of Page Link type ACF fields.

== Installation ==

1. Upload the `distribute-acf-images` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

Not at the moment.
